Esta pasta de código contém um programa C++ que realiza uma série de cálculos matemáticos envolvendo as variáveis a, b, c e r1. O programa realiza as seguintes operações:

Imprime os valores de a, b, c e r1 na saída padrão (console).
Esta pasta de código pode ser útil para fins de aprendizado, demonstrando como realizar cálculos simples em C++ e usar operadores aritméticos e de comparação.
