 Esta pasta contém um código em C++ que calcula as raízes de uma equação quadrática da forma ax^2 + bx + c = 0. O programa solicita os valores dos coeficientes a, b e c ao usuário e, em seguida, utiliza a fórmula quadrática para calcular as raízes. As raízes são exibidas como saída, sendo x e y os valores das duas raízes possíveis. O código utiliza a biblioteca math.h para calcular a raiz quadrada. Este programa é útil para resolver equações quadráticas em matemática e engenharia.
